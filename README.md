# HisaabMate

HisaabMate is an Android application designed for managing personal finances (Hisaab Kitab). It helps users track their transactions, manage accounts, and monitor their financial goals.

## Features

- **Transaction Tracking**: Record income and expenses.
- **Account Management**: Manage multiple bank accounts and wallets.
- **Financial Goals**: Set and track financial targets.
- **Dashboard**: Overview of your financial health.

## Technologies Used

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Database**: Room
- **Architecture**: MVVM with Clean Architecture principles

## Setup

1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle project.
4. Run on an emulator or physical device.
